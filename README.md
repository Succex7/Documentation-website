# Documentation-website
A simple documentation website like a mini W3Schools-style Docs

## Objectives
-Structured layouts for real documentation <br>
-Content organization <br>
-Internal linking

## Techhnologies
-HTML5 <br>
-CSS3

## Page structure
-Header <br>
-nav tags with top-level links <br>
-aside tags containing table of contents <br>
-main tags with: 
<li>Multiple documentation  section using section tags</li>
<li>code examples using pre and coe tags</li>
<li>Internal links using id</li>

## CSS Styling
-Different background color per section <br>
-Clear spaccing and typography
